```rust
fn resolve_channels(ctx: &DispatchContext<'_>) -> Option<String> {
    if let Some(uc_id) = ctx.use_case_id {
        if let Ok(persona) = persona_repo::get_by_id(ctx.pool, ctx.persona_id) {
            if let Some(dc_str) = persona.design_context.as_deref() {
                if let Ok(dc) = serde_json::from_str::<serde_json::Value>(dc_str) {
                    if let Some(uc) = dc.get("use_cases").and_then(|v| v.as_array())
                        .and_then(|arr| arr.iter().find(|u| u.get("id").and_then(|v| v.as_str()) == Some(uc_id)))
                    {
                        if let Some(nc) = uc.get("notification_channels") {
                            if nc.is_array() && !nc.as_array().unwrap().is_empty() {
                                return Some(nc.to_string());
                            }
                        }
                    }
                }
            }
        }
    }
    ctx.notification_channels.map(|s| s.to_string())
}
```

Cache the resolved string on DispatchContext if the DB roundtrip bothers you.

## 9. One-paragraph model of the architecture

Persona is an identity + shared state (tools, core memories, governance). Capabilities (use cases) are first-class units stored as JSON inside `design_context.useCases[]`. Each capability has an `enabled` flag, a trigger, event subscriptions, optional `notification_channels`, optional `model_override`, and a `sample_input`. Runtime (C1): `assemble_prompt` always renders a `## Active Capabilities` section filtered by `enabled != false`, and `execute_persona` auto-expands `use_case_id` into `input_data._use_case` before spawning. Session cache (C1) includes an active-capability fingerprint so toggles invalidate warm sessions. UI (C3): Use Case tab surfaces a Power toggle + Simulate button per capability; cascade dialog previews affected triggers/subs/automations before disable. Triggers/automations (C4): attached to capabilities at creation time. Messages/reviews/memories (C5 — **your job**): inherit `use_case_id` from the execution row; memories also filter on it at injection time. Lab per-capability (C6): RFC first, then landed.
