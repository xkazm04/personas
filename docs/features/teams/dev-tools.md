# Dev Tools → moved

The canonical Dev Tools reference now lives at
**[`../plugins/dev tools/dev-tools.md`](../plugins/dev%20tools/dev-tools.md)**.

This page used to hold a second, parallel copy of the Dev Tools docs. After the
**2026-07 Projects-IA consolidation** — the top-level *Teams* section was
rebranded **Projects**, and project **Management**, **Goals**, **Lifecycle**, and
**Competition** moved out of the Dev Tools plugin into that section — the two
copies drifted. The plugin doc is the maintained one; `feature-doc-map.json`
points here-no-longer and the Stop hook now watches the plugin doc.

For project-level surfaces (Manage / Goals / Lifecycle / Competition / Factory),
see the **Projects** section docs; for the dev-automation surfaces (Overview,
Observability, Context Map, Idea Scanner, Idea Triage, Task Runner, Fleet), see
the plugin doc linked above.
