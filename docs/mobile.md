  1. Start dev mode: npm run dev
  2. Press Ctrl+Shift+M — page reloads in mobile layout mode
  3. A cyan "MOBILE PREVIEW" badge appears in the top-right corner
  4. You see exactly what Android users see: 4 sidebar modules, collapsed rail, overlay drawer, single-column grids,
  card-based execution list, Remote Control button, etc.
  5. Press Ctrl+Shift+M again to toggle back to desktop layout
  6. Resize your browser to ~390px wide for the most realistic phone simulation (Chrome DevTools device toolbar works too)
